Smart Crash Detection for Barangay Sta. Maria

DisgrasEye is a Computer Vision-based system designed to monitor Barangay Sta. Maria through existing CCTV cameras, automatically detecting vehicular accidents and sending real-time alerts to barangay officials and authorities.

Admin Dashboard Explore Features

System Features

Enhancing public safety and traffic response in Barangay Sta. Maria.

CCTV Monitoring

Uses Barangay Sta. Maria’s CCTV feeds for real-time traffic monitoring.

Accident Detection

Detects and logs crash events automatically using computer vision.

Instant Alert System

Notifies barangay officials and nearby responders immediately.

Crash Log Storage

Stores each detected event with timestamp, location, and severity.

Incident Analytics

Displays trends and hotspots for better road management.

Admin Control

Secure admin access for authorized Sta. Maria personnel.

Technology Stack

Developed with efficient and reliable modern tools.

CCTV Integration
Django Framework
SQLite + Cloud Backup
